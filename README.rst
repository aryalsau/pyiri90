=============
iri90-python
=============

IRI90-international reference ionosphere in Python

Installation
============

.. code:: bash

    git clone --depth 1 --recursive https://github.com/scienceopen/pyiri90
    pip install -r requirements.txt
    make -f Makefile.f2py
